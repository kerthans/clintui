## Using Intent CLI
Start a new project by running the following command:
<Installation command='npx @intentui/cli@latest init' />

Intent will guide you through a few setup questions:
```
✔ No setup project detected. Do you want to start a new project? (Y/n) Yes
✔ What language do you want to use? TypeScript
? Which framework do you want to use?
  Laravel
❯ Next.js
  Remix
  Vite
```

## Using Next.js Starter Kit
Kickstart your project with the [Next.js Starter Kit](https://github.com/intentuilabs/next.js), pre-configured and ready to use:
<Installation command='npx create-next-app@latest -e https://github.com/intentuilabs/next.js app'/>

## Source Code
To install Intent manually, clone the latest version from the [GitHub repository](https://github.com/intentuilabs/next.js).
